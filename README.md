📈 StockVision – Smart Stock Market Dashboard

StockVision is a professional responsive stock market dashboard website developed using HTML5, CSS3, JavaScript, Chart.js, and API integration.

🚀 Features

* Live Stock Market Dashboard
* Cryptocurrency Tracking
* Real-Time API Integration
* Portfolio Tracker
* Currency Converter
* Interactive Charts using Chart.js
* Responsive Multi-Page Design
* Dark/Light Mode
* Dynamic Image Slider
* Modern FinTech UI
* Contact Form Validation
* Market News Section
* Glassmorphism Effects
* Mobile Responsive Layout

🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Chart.js
* Alpha Vantage API
* CoinGecko API

📂 Pages

* Home Page
* Market Dashboard
* Contact Page

📊 APIs Used

* Alpha Vantage API
* CoinGecko API

🎯 Project Objective

To develop a modern responsive stock market dashboard capable of displaying financial data, cryptocurrency analytics, portfolio management, and dynamic user interactions using frontend web technologies.

📱 Responsive Design

The project is fully responsive across:

* Desktop
* Tablet
* Mobile Devices
